## Usage

1. Open `index.html` in a browser.
2. Enter a certificate ID from `SSA-2026-101` to `SSA-2026-110`.
3. View the corresponding student name, course, and issue date.

## Notes

This is a front-end mock demo and does not use a real backend database. It is for training and presentation purposes only.
